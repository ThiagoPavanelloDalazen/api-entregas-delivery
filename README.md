# API-Entregas

## 🚀 Como rodar o projeto

### Pré-requisitos
- Node.js
- npm

### Instalação

Clone o repositório:
```bash
git clone https://github.com/ThiagoPavanelloDalazen/API-Entregas.git
cd API-Entregas
```

Instale as dependências:
```bash
npm install
```

### Dependências utilizadas
- Express 4.19.2
- TypeScript 5.5.4
- tsx 4.16.2
- Zod 3.23.8
- express-async-errors 3.1.1
